# Rock, Paper, Scissors

This is a simple browser-based implementation of the classic Rock, Paper, Scissors game.

## Overview

The game allows a player to compete against the computer. The game logic is handled in JavaScript, and interactions occur via browser prompts and alerts.

## How to Run

1.  Clone or download this repository.
2.  Open the `index.html` file in any modern web browser.

## How to Play

1.  Click the **Start Game!** button on the page.
2.  A prompt will appear asking you to choose between **ROCK**, **PAPER**, or **SCISSORS**.
3.  Type your choice and click **OK** (or press Enter).
    *   If you make an invalid choice, a default choice (ROCK) will be selected for you.
4.  The computer will make its choice.
5.  An alert will display the winner: You, the Computer, or a Tie.

## Technologies

*   HTML5
*   JavaScript
